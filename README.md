# 空间分析课程 · 交互学习网页（Lect.4—Lect.13，v2）

这是一个本地运行的互动学习网站，基于 Lect.4—Lect.13 课件重组。v2 重点解决“公式不突出、英文课件阅读阻碍、题目偏少、案例偏课件内”的问题。

## 这次新增 / 优化

- **公式与计算**：每讲整理 2—3 个关键公式，显示公式、变量含义、判断逻辑、常见误解和计算思路。
- **中文导读**：将课件的主要页段改写为中文学习线索，并增加英汉术语表；不是逐字机械翻译。
- **真实案例**：加入 NASA、USGS、Esri 官方材料与本网站原创图解，避免只看英文 PPT 截图。
- **更多自测**：每讲 8 道选择题 + 2 道可输入答案的计算题，共 100 道学习题。
- **排版优化**：关键公式改为大字号深色公式卡；阅读区、提示区、案例区与计算区的视觉层级更明显。

## 运行方式

1. 解压整个压缩包到一个文件夹。
2. 双击 **`Start_Website.bat`**。
3. 保持弹出的黑色命令行窗口开启。
4. 浏览器会自动打开；若没有自动打开，在浏览器输入：

```text
http://127.0.0.1:8000/index.html
```

如果 8000 端口被占用，可关闭占用该端口的本地服务后再运行。

> 只要不关闭命令行窗口，网站就可继续使用。关闭窗口后，网页服务会停止。

## 网页包含的内容

- Lect.4—Lect.13，每讲 4 个系统化的中文详细讲解单元。
- 每讲 2—3 张“公式与计算”卡片。
- 每讲课件主要结构的中文导读和英汉术语表。
- 每讲 6 张原始课件关键页截图，可点击放大。
- 10 个离线互动实验：VMR、Moran’s I、OLS/GWR 直觉、PPDAC、K-means、随机森林、时空立方体、CNN 卷积、并行处理和超分辨率。
- 每讲 8 道选择题与 2 道计算题。
- 原始 PDF 放在 `sources/`，可从网页顶部按钮直接打开。
- 本地笔记：仅保存在当前浏览器，不上传。

## 文件结构

```text

```

## 重要说明

- 互动实验和计算题用于建立直觉，不替代 ArcGIS Pro、GeoDa、R 或 Python 的正式统计计算。
- 外部案例图仅用于本地教学展示，来源和图像署名见 `REAL_CASE_SOURCES.md`。
- 做作业或论文时，应单独报告研究区、空间尺度、参数设置、数据来源、显著性检验、验证方案和方法局限。
